# Introduction

The **mp_carentan_fix** map fixes several bugs in the original **mp_carentan** map for Call of Duty 2.

## Download
- Actual version:
	- 2022/07/28 - <b><a href="https://github.com/eyza-cod2/mp_carentan_fix/raw/main/mp_carentan_fix_v1_test.iwd">Carentan FIX v1 test</a></b>


## Installation
- Download <b><a href="https://github.com/eyza-cod2/mp_carentan_fix/raw/main/mp_carentan_fix_v1_test.iwd">Carentan FIX v1 test</a></b> and move it into main folder.
- Change map on the server via command **/rcon map mp_carentan_fix_v1_test**


- Notes:
	- This is first version of the map. Final map name will be mp_carentan_fix
	- Report bugs to **eyza** - discord: eyza#7930
